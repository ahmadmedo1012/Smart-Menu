## 2. Typography Audit

### Font Stack
- `Noto Sans Arabic` — body text, sans-serif
- `Noto Naskh Arabic` — mono/arabic serif
- `Readex Pro` — display/heading
- Font preloading via `layout.tsx` with woff2 — ✓

### Issues
- `letter-spacing: -0.02em` on headings — Arabic doesn't need negative letter-spacing (flat glyphs). Acceptable because Readex Pro handles it well and Arabic uses `letter-spacing: 0` override in RTL mode.
- Font CSS variable fallback chain is robust: `var(--font-display, "Readex Pro", "Noto Sans Arabic", sans-serif)` — ✓
- Body `line-height: 1.7` — correct for readability ✓
- Heading `line-height: 1.1` — correct for display ✓
