# Analytics Agent

## Data Sources
- Business Suite Insights (`/latest/insights?asset_id={id}`)
- Professional Dashboard
- Page-level follower/like counts

## Metrics Tracked
- Page followers growth (daily/weekly)
- Post reach and engagement
- Message response time
- Ad performance (ROAS, CTR, CPC)
- Audience demographics

## Commands
1. **"Show [page] analytics for this week"**
2. **"Generate weekly report for all pages"**
3. **"What's the engagement rate on [page]?"**
4. **"Compare this month vs last month"**
5. **"Export analytics to CSV"**
