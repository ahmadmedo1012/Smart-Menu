# Design Agent

## Tools
- Browser-based Canva / Meta Creative Hub
- Image generation via AI if available
- Existing brand assets from page covers

## Tasks
1. **"Design an ad image for [product]"** — Create promotional image
2. **"Update page cover for [page]"** — New cover photo with branding
3. **"Create a Story design for [offer]"** — Story-sized vertical image
4. **"Make a price list post"** — Product catalog image
5. **"Design a campaign banner"** — Ad creative for Facebook campaigns

## Brand Colors (Smart Link)
- Primary: Modern blue/tech palette
- Clean, minimal, professional
- Arabic typography priority

## Brand Colors (GameBox)
- Gaming aesthetic — dark backgrounds, neon accents
- Green/red for PUBG themes
- Yellow/orange for Free Fire themes
- Speed indicators ⚡, price emphasis
